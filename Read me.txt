README — Запуск Telegram-бота (Task Bot)

Кратко: это пошаговая инструкция для пользователя без опыта программирования: 
как подготовить компьютер, вставить токен и запустить бота двойным кликом (Windows)

Требования (очень важные)

-Python 3.10 или новее (рекомендуется 3.10/3.11). Если у вас Windows — при установке обязательно отметьте "Add Python to PATH".

-Подключение к интернету (бот использует polling).

-Токен от BotFather (инструкция ниже).

Как получить токен у BotFather (шаги в Telegram)

Откройте Telegram и найдите пользователя @BotFather.

Нажмите /start (или введите /start).

Нажмите кнопку /newbot и следуйте инструкциям: укажите имя бота и его username (username должен заканчиваться на bot, например mytaskbot).

После создания BotFather пришлёт токен вида 123456789:ABCDefGhIjK... — скопируйте этот текст.

Вставьте токен в файл token.txt (только токен, без кавычек и переносов строк): 123456789:ABCDefGhIjK...

Запустите run.bat
